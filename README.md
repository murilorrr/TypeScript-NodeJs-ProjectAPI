# TypeScript-NodeJs-ProjectAPI
An introduction to TypeScript with a simple project  API on NodeJS for beginners (like me)
​
## Docker
​
To start the container
```sh
$ docker-compose up -d --build
```
​
To verify running process docker
```sh
$ docker ps
```
​
To rerun
```sh
$ docker-compose down
$ docker-compose up -d
```
